# Lokala figurbilder

Lägg egna PNG/WebP/JPEG/SVG-bilder här och referera till dem i ett figurpaket med till exempel:

```json
{
  "id": "min-figur",
  "name": "Min figur",
  "description": "Kort beskrivning.",
  "imagePath": "./assets/creatures/min-figur.png"
}
```

Bilderna ska vara små nog för mobil/PWA-cache. Transparenta PNG/WebP-filer fungerar bäst.
